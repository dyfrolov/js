Способы подкл CSS (убывание приоритета): 
1. inline атрибут style="color:red;font-style:"
2. внутри тега <style> (обычно внутри <head></head>)
3. внешний <link rel="stylesheet" href="">


Селектор - шаблон, указывающий множество, для которых применяются свойства.
selector{
	property:value;
	...
}

группировка селекторов:
selector1, selector2 {
	property:value;
	...
}


Селекторы:
* - универсальный селектор, применяется ко всем элем на странице
tag - указываются тэги.
.class - может для разых типов тэгов
#id - уникальный (для различных контекстных селекторов или селекторы потомков)
селекторы псевдоклассов, ....
селекторы атрибутов


2 основных принципа CSS:
 - наследование (какие-то элементы наследуют стили родителей, для тех, которые не наследуют по умолчанию, указывают property:inherit;)
 - каскадирование  - объединение свойств элемента, определенных в разных селекторах. При коллизиях(конфликтах) - приоритеты:
	1. стиль родителя
	2. стиль подключаемого файла стилей
	3. стиль в <style> 
	4. inline
	5. color:blue !important;
